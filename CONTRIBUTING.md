# Contributing to This Portfolio

This portfolio is open source and you're welcome to use it as a template or contribute improvements!

## Using This as a Template

1. Fork the repository
2. Update `_config.yml` with your information
3. Replace example content in `_posts/` and `_projects/`
4. Add your own images to `assets/images/`
5. Customize colors in `tailwind.config.js`
6. Deploy to your own GitHub Pages

## Reporting Issues

If you find bugs or have suggestions:

1. Check if an issue already exists
2. Create a new issue with a clear description
3. Include steps to reproduce (if applicable)

## Submitting Changes

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/improvement`)
3. Make your changes
4. Test thoroughly (`make build` and `make serve`)
5. Commit with clear messages
6. Push to your fork
7. Create a Pull Request

## Code Style

- Use 2 spaces for indentation
- Follow existing file structure
- Keep Liquid templates readable
- Use semantic HTML
- Write accessible markup
- Test dark mode

## Testing

Before submitting:

```bash
make clean
make install
make build
make serve
```

Verify:
- [ ] Site builds without errors
- [ ] All links work
- [ ] Images load correctly
- [ ] Dark mode works
- [ ] Mobile responsive
- [ ] Keyboard accessible

Thank you for contributing!
